# thiyagarajan-a.github.io

Portfolio website for Theo ( Thiyagarajan Anandan ).

## Structure

- `index.html` - home + positioning + impact highlights
- `about.html` - mission, approach, leadership timeline
- `projects.html` - case study index
- `project-*.html` - detailed case study pages
- `blog.html` + `blog-*.html` - writing pages
- `resume.html` - HTML resume summary + downloadable PDF
- `contact.html` - contact form (mailto flow) + direct links
- `styles/main.css` - shared design system and responsive layout
- `scripts/main.js` - nav, reveal animation, active route, contact form behavior
- `assets/Thiyagarajan_Anandan_Resume_Oct_2025.pdf` - resume PDF

## Local preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

This repo is intended for GitHub Pages from the `main` branch.
Pushing changes to `main` publishes the updated site.
